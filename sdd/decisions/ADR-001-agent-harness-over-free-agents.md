# ADR-001: Use Agent Harness Over Free-Running Agents

## Status

Accepted

## Context

المشروع يحتوي توقعات، مصادر بيانات، إحصاءات، مراجعة، وتطور. تشغيل وكلاء مستقلين بدون harness سيؤدي إلى فقدان السياق وتضارب القرارات.

## Decision

نستخدم `AgentHarness` كمنسق مركزي. كل وكيل له عقد واضح ومدخلات ومخرجات محددة.

## Consequences

- أسهل في الاختبار.
- أسهل في مراجعة سبب التوقع.
- أقل عرضة لانهيار السياق.
- أي وكيل جديد يجب أن يدخل ضمن العقد.
